Usage

Features

Image Import

Load any image from your filesystem to use as your meme template with just a click.

Draggable Text

Place classic impact-style meme text anywhere on the canvas.

Built-in Templates

Pick from a library of popular meme templates or import your own to get started instantly.

Layers

Import additional images as layers on top of your base template.

Different Export Format

Support 3 file formats. PNG, JPG, and WebP.

Native GNOME Design

Built the latest Libadwaita.

Installation

Get Memerist

Choose the installation method that works best for your distribution.

Flathub Recommended

The easiest way.

flatpak install flathub io.github.vani_tty1.memerist

Fedora COPR

Native RPM package for Fedora users via the COPR build system.

# Enable the COPR repo
sudo dnf copr enable vaniiiiii/memerist
sudo dnf install memerist
Open COPR Page

Build from Source

Build it yourself.

# Install dependencies:
gtk4-devel libadwaita-devel meson ninja-build libepoxy-devel blueprint-compiler
# Clone and build
git clone https://github.com/vani-tty1/memerist.git
cd memerist
# builds and launches
make run
